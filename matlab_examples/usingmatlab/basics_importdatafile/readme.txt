Example: wave_intro
Demonstrates the following features

1. Using C-like file streaming to write output data
2. Data import from ascii using the load function
3. Example of cell array
4. Plotting array of matrices into  surface sub plots


INSTRUCTIONS
1. From matlab set current directory to import.
2. Run  the script wav2dtest.m by (type wav2dtest).
3. Load the ascii data by running the script wav2d_load.m (type wav2d_load). 
4. Plot the imported data by running the script wav2d_plot.m (type wav2d_plot).
